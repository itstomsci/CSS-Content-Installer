Select Directory
To correctly select the place of your Garry's Mod folder, you will need to follow these steps:
  1. Open Steam and click on "Library" in the top left.
  2. Find "Garry's Mod" in the list on the left side and right-click on it.
  3. Hover over "Manage" and click "browser local files".
  4. This should then open File Explorer at the correct location of where your Garry's Mod is installed.
  5. Click on where it shows the folder path (should include "Steam > steamapps > common > GarrysMod").
  6. Now you need to copy the whole path to your clipboard.
  7. Next, click the "Select File (hl2.exe)" at the bottom right.
  8. Click on where it shows the folder path and delete the current path.
  9. Now paste the path from your clipboard that you copied earlier.
  10. Lastly, select the hl2 (hl2.exe) file and click "Open" in the bottom right.
